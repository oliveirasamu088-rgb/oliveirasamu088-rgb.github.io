Compatibilidade do Site

Para garantir que você tenha a melhor experiência de navigation, animações fluidas e leitura otimizada, verifique os requisitos mínimos recomendados para o seu dispositivo.

Dispositivos Recomendados

O site foi totalmente otimizado para rodar sem falhas em:

  • Computadores/Notebooks: Windows 10/11, macOS ou Linux.
  • Smartphones Modernos: Android (versão 8.0+) e iOS (iPhone 7 ou superior).
  • Navegadores: Google Chrome, Safari, Mozilla Firefox, Opera e Microsoft Edge atualizados.

Onde Podem Ocorrer Falhas?

Dispositivos antigos ou navegadores obsoletos que não dão suporte a CSS moderno (Flexbox/Grid):

  • Internet Explorer: Falha crítica. O layout quebrará completamente (não suporta variáveis CSS e Grid).
  • Sistemas Antigos: iPhones com iOS 11 ou inferior, e Androids anteriores à versão 6.0 (erros de renderização de fontes e ícones).
  • Smart TVs Antigas: Navegadores integrados de TVs fabricadas antes de 2018 costumam travar as animações de flutuação (`@keyframes float`).
